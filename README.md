## Objetivo

Praticar o fluxo básico de trabalho com Git: clonar repositório, editar, fazer commit e enviar um Pull Request.

## Como fazer a atividade

1. **Faça o clone do repositório**
   ```bash
   git clone <url-do-repositorio>
   cd moro-atv1-github
   ```

2. **Abra o projeto no VS Code**
   - Abra a pasta do projeto no Visual Studio Code (ou Cursor).

3. **Adicione seu nome na lista da sua turma**
   - Abra o arquivo `index.html`.
   - Na seção da sua turma (1º, 2º ou 3º Ano), adicione uma nova linha na lista com seu nome, por exemplo:
     ```html
     <li>Seu Nome Completo</li>
     ```
   - Salve o arquivo.

4. **Faça commit com a mensagem solicitada**
   ```bash
   git add index.html
   git commit -m "Adicionado nome do aluno (Seu Nome)"
   ```

5. **Envie um Pull Request**
   - Envie suas alterações para o seu fork (`git push`).
   - No GitHub, abra um **Pull Request** do seu fork para o repositório original.
